Install Files

  1. Download the application files to your computer.
    • Option 1 (easier): click here to download
    • Option 2 (for developers): clone files from the GitHub repository
  2. Open up a command line application (e.g. Command Prompt on Windows, Terminal on Mac)
  3. Navigate to the folder where you downloaded the application files (e.g. chirpDatabase2-master. Type ls in the command prompt to list the files and directories in the folder. You should see app, bin, src, and web folders, among others (this will verify you are in the right place).
  4. Install Composer in this folder, if it is not already installed globally on your computer. This program is needed to download additional files that are required for the application to function.
  5. Run php composer.phar install to install the remaining files needed to run the application (this will download files into a vendor folder in the project directory). Composer will install files for a few minutes, then ask you for 'parameters' - information needed for the application to access your database. Enter these values from the database you created above when prompted, or hit enter to accept the default value (you will at least need to enter the database name, user and password). The application also has a 'forgot password' functionality that will email users who forget their passwords. Enter gmail credentials (details needed here...)
  6. Google and fix any errors you come across
  7. Run php bin/symfony_requirements in the command prompt (type this and then hit Enter on your keyboard) to verify your computer can run the code; fix any errors.
