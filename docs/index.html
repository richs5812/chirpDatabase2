Installation Instructions

The following are instructions on how to set up the Brightmoor Connection database. This is a web application built with the PHP framework Symfony.

Set Up Web Server

  • Easy option that works on Windows, Mac and Linux: download and install XAMPP. Make sure you install at least Apache, MySQL, PHP, and phpMyAdmin during the installation process. If prompted, allow XAMPP services to run on private networks only (recommended security setting)

Create Database

You will need to create a MySQL database on your server. There are many ways to do this; I have documented how using XAMPP and phpMyAdmin below:

  • open XAMPP
  • Click Admin in the MySQL row
  • This will open phpMyAdmin in your web browser, which allows you to administer your databases. Click Databases in the top menu, enter a name for your database in the Name field, then click Create.
  • You have successfully created your database; now we will create the database user. Click Privileges in the top menu.
  • Click Add user account
  • Enter a user name; type localhost in the Host name field; enter a password. Scroll down and under 'Database for user account' check the 'Grant all privilieges on database your_database_name' box. Scroll to the bottom of the page and click Go to create the user account.

Save the database name, user name and password for below, when you will need to enter this information for the application to connect to the database

Install Files

  1. Download the application files to your computer.
    • Download the zip file, or clone from the GitHub repository
  2. Save the files in a location accessible by your web server. If you are using XAMPP, you will need to save these files inside the xammpp/htdocs folder.
  3. Open up a command line application (e.g. Command Prompt on Windows, Terminal on Mac; there is also a built-in shell tool in XAMPP), and navigate to where you saved the files; enter the top level folder (e.g. chirpDatabase2-master). A listing of the files - type ls (Mac/Linux) or dir (Windows) in the command prompt - should show app, bin, src, and web folders, among others if you are in the right place.
  4. Install Composer in this folder, if it is not already installed globally on your computer.
  5. Run php composer.phar install to install the remaining files needed to run the application (this will download files into a vendor folder in the project directory). Composer will install files for a few minutes, then ask you for 'parameters' - information needed for the application to access your database, as well as the gmail account needed for the password reset function. Enter database values from the database you created above when prompted, or hit enter to accept the default values (you can fill in or edit these values later in the parameters.yml file located in the app/config directory). Enter gmail credentials (details needed here...)
  6. Google and fix any errors you come across. You can also refer to Symfony's deployment guide if you run into any errors.
  7. Run php bin/symfony_requirements in the command prompt (type this and then hit Enter on your keyboard) to verify your computer can run the code; fix any errors.
