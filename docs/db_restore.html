Restoring the database from a backup file

  1. Open the XAMPP Control Panel (click the orange logo on the dock)
  2. Click Admin in the MySQL row to open phpMyAdmin.
  3. Click on chirpdatabase in the left hand column.
    4. view full-size image
  4. Click Import on the top menu.
    5. view full-size image
  5. Click the Browse button.
    6. view full-size image
  6. Click the database backups folder in the Favorites menu. You can also access this folder by going to Documents > database backups.
    7. view full-size image
  7. Select the backup file you want and click Open. You can sort the backups by date, or read the file name to see the date and time the file was created.
    8. view full-size image
  8. Once you have selected the file, scroll to the bottom of the phpMyAdmin screen and click 'Go'.
    9. view full-size image
  9. After a few moments, you should see a success message. The import is complete; you can close the phpMyAdmin window and resume using the database. You will notice that the data is now from the backup file you just imported.
    10. view full-size image
